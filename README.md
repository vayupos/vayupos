#  VayuPOS – Restaurant Point of Sale System

VayuPOS is a full-stack Restaurant Point of Sale (POS) system designed to manage restaurant operations such as billing, menu management, customers, and offers efficiently.

This project follows a modular structure with separate frontend and backend folders to support clean architecture and team collaboration.

---

## Project Structure

VayuPos/
│
├── backend/ # Backend (FastAPI)
├── frontend/ # Frontend (React)
└── README.md # Project overview



---

##  Tech Stack

### Frontend
- React.js
- JavaScript
- HTML, CSS

### Backend
- FastAPI
- Python
- PostgreSQL
- SQLAlchemy
- Alembic

---

##  Features

- POS Billing System
- Menu Management
- Customer Management
- Coupons & Offers
- REST API Integration

---
